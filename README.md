# Blog Platform

A responsive blog platform built using **HTML, CSS, and JavaScript**.

## Features
- Create, edit, and delete blog posts
- Read More / Read Less functionality
- Pagination
- Dark mode
- Responsive design
- Data persistence using LocalStorage
